# Contributing to WorrkFree Agile

First off, thank you for considering contributing to WorrkFree! It's people like you that make open source such a great community.

## How to Contribute

### 1. Fork and Clone
1. Fork the repository on GitHub.
2. Clone your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/worrkfree-agile.git
   cd worrkfree-agile
   ```

### 2. Set Up Your Environment
1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy the example environment file and add your keys:
   ```bash
   cp .env.example .env
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### 3. Make Your Changes
- Create a new branch for your feature or bugfix:
  ```bash
  git checkout -b feature/my-awesome-feature
  ```
- Make your changes and test them locally.
- Run the linter to ensure code quality:
  ```bash
  npm run lint
  ```

### 4. Submit a Pull Request
- Commit your changes with a descriptive commit message.
- Push your branch to your fork:
  ```bash
  git push origin feature/my-awesome-feature
  ```
- Open a Pull Request against the `main` branch of the original repository.
- Ensure you fill out the provided Pull Request template.

## Code of Conduct
Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
